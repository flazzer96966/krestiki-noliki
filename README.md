===========================================
         КРЕСТИКИ-НОЛИКИ v1.0
         Игра для двух игроков
===========================================

[ОПИСАНИЕ]
-------------------------------------------
Классическая игра "Крестики-нолики" на поле 3x3.
Два игрока по очереди ставят X и O, пытаясь выстроить
три своих символа в ряд. Создана в рамках выполнения модуля Python от А до Я: для практиков.
В репозитории расположен исполняемый файл main.exe с игрой крестики-нолики и файл исходного кода main.py.

[ПРАВИЛА]
-------------------------------------------
1. Игроки ходят по очереди
2. Игрок X делает первый ход
3. Побеждает тот, кто первым выстроит 3 своих символа в ряд
4. Если поле заполнено, а победителя нет - объявляется ничья

[КАК ИГРАТЬ]
-------------------------------------------
Запустите программу и выберите в меню:
  1 - начать новую игру
  2 - посмотреть правила
  3 - выйти

При ходе вводите координаты через пробел:
  "строка столбец"

Примеры:
  "1 1" - левая верхняя клетка
  "2 3" - вторая строка, третий столбец

Вид поля:
    1   2   3
1   |   |
   --------
2   |   |
   --------
3   |   |

[СИСТЕМНЫЕ ТРЕБОВАНИЯ]
-------------------------------------------
• Windows 7/8/10/11
• Любой процессор
• От 64 МБ ОЗУ
• 5 МБ свободного места

[УСТАНОВКА]
-------------------------------------------
Вариант 1:
  Просто запустите файл крестики-нолики.exe

Вариант 2:
  1. Установите Python 3.6+
  2. Запустите: python tic_tac_toe.py

[СОВЕТЫ]
-------------------------------------------
• Начинайте с центральной клетки (2 2)
• Блокируйте ходы противника
• Создавайте "вилки" (два пути к победе)
